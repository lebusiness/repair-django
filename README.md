# repair

## Create virtual env
```
python3 -m venv venv
```

### Activate virtual env (on Windows)
```
venv\Scripts\activate.bat
```

### Activate virtual env (on MacOs, Linux)
```
source venv/bin/activate
```

### Installing
```
pip3 install django
```
```
pip install --upgrade pip
```
```
python -m pip install Pillow
```

### Run server
```
python manage.py runserver
```
